Mac实用软件推荐

发表于 2015-08-16   |   分类于 Tech   |     |   阅读次数

这篇文章主要是记录我在Mac上遇到的一些好用的软件，很多非常常见的我就没有列出来了，当我遇到了新的我会在这里更新。

  • zsh和oh-my-zsh：这个不算是mac的软件，我在任何一个unix-like系统中的默认shell都会改为zsh并用oh-my-zsh进行配置，效率大大提升。
  • iterm2：
  • genymotion：一个速度非常快的android开发时用到的设备模拟器，完爆google的模拟器。
  • scroll reverser：可以很方便的分别选择鼠标和触控板是否翻转滑动模式，这样鼠标和触控板都能以我们习惯的方式上下滚动屏幕啦。
  • homebrew和brew cask：前者是mac上非常好用的包管理软件，后者是基于homebrew构建的可以安装Mac Application的软件，免去了安装Mac软件时在网上搜索、下载和拖拽至Application目录的过程。
  • GoAgentX：goagent在Mac上的GUI，用于翻墙，不过最近goagent总是不太稳定。
  • Mou和Macdown：两者都是Mac上的Markdown编辑器，前者开始收费了，后者是模仿前者的开源版本
  • better touch tool：十分好用的手势自定义软件，可以自定义触摸板，Magic Mouse和普通鼠标的各种手势对应的功能。
  • cheatsheet：按住Command键就能显示当前程序所有的系统默认快捷键。
  • licecap：录制gif格式图片的一个软件，可以设定分辨率和最大帧率，软件支持自动识别设定实时真率，使用起来很流畅。
  • pycharm：Python IDE，我觉得是Mac上开发python最好用的软件，有社区版和专业版，社区版免费，专业版支持各种框架。喷脑浆公司出了一系列的针对各种编程语言的IDE，体验很一致也很不错。
  • SizeUp：Mac自带的窗口控制太弱了，这个软件可以定义各种快捷键实现各种方式的窗口resize，非常方便。
  • VisualDiffer：一个可以比较两个文件或者文件夹之间差异的软件。同类的软件还有一个跨平台的Beyound Compare
  • Vox：一个轻量级的但是功能强劲的音乐播放软件，支持各种音频格式，包括无损的各种格式，软件UI的感觉也不错。
